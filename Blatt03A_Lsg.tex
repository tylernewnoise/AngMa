	\textbf{Aufgabe 2}
	
	Produkte: Perfekt Rasen = $ x_1 $; Super Rasen = $ x_2 $\\
	
	$(P_1) \text{ max} \{\ 5x_1 + 8x_2 \mid \begin{matrix}
	2x_1 + 3x_2 \leq 1500\\
	x_1 + 2x_2 \leq 1200\\
	2x_1 + x_2 \leq 1300\\
	x_1, x_2 \geq 0
	\end{matrix}
	\ \}$\\\\
